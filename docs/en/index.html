German COVID-19 Vaccine Data

The goal of this dashboard is to document and visualize the progress of COVID-19 vaccinations in Germany.

The vaccination data is provided by the Robert Koch Institute (RKI). The data presented here was retrieved from this public ARD Data GitHub repository.

Population data for the federal states is taken from the Federal Statistics Office (Statistisches Bundesamt).

The graphics in this dashboard show the vaccination date which is usually the day before the reporting date. However, sometimes there are delays and federal states do not report data until later. Therefore, please note the comments in the tooltips of the graphics (notes are in German) NOTE: The notes are currently not being updated! Check the raw .xlsx files if you want to look at them: ARD Data GitHub repository.
