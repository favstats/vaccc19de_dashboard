German COVID-19 Vaccine Data

The goal of this dashboard is to document and visualize the progress of COVID-19 vaccinations in Germany.

The vaccination data is provided by the Robert Koch Institute (RKI). The data presented here was processed with the help of the R package vaccc19de.

Population data for the federal states is taken from the Federal Statistics Office (Statistisches Bundesamt).

The graphics in this dashboard show the vaccination date which is usually the day before the reporting date. However, sometimes there are delays and federal states do not report data until later. Therefore, please note the comments in the tooltips of the graphics (notes are in German). A dataset with all previous comments can be viewed here.

⚠️ Note: The RKI started to differentiate between first and seccond vaccination in its data on 2021-01-18. This resulted in a couple of changes in the structure of their Excel file (namely, more columns, multi-row column headers) which might not look drastic to the human eye but are significant to the code that we have written to read in the excel and put it into a better, more consistent format. Hence, we currently can’t update the dashboard. We are working on fixing this issue. ⚠️
